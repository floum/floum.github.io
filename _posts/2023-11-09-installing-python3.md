# Installing python3

**Disclaimer :** This method has been tested on Ubuntu LTS 22.04.

## Requirements

- Super User rights (sudo or log in as root)

## Installing Python3

- Update your system

```
sudo apt update
sudo apt upgrade
```

- Install python

```
sudo apt install python3 python-is-python3
```

- Check your python installation

```
which python3

#=> /usr/bin/python3

python -V

#=> Python 3.10.12
```
