Ruby Arrays

Arrays are ordered, integer-indexed collections of any object.

Defining arrays

Arrays of strings can be defined in many ways :

The classic way™ (aka the literal constructor)

[1, 2, 3]

The object oriented way : Array.new

ruby Array.new #=> [] Array.new(2) #=> [nil, nil] Array.new(2, 4) #=> [4, 4]

Be aware that mutable objects (like hashes, strings, arrays…) should be instantiated with a block like this :

Array.new(3) { Hash.new }

See this pitfall in action :

ruby hash = Hash.new array = Array.new(3, hash) #=> [{}, {}, {}] array[0][:key] = [:value] array #=> [{key: :value}, {key: :value}, {key: :value}]

Using Arrays

Now that we have know more about creating arrays, let's have some useful interactions with them :

 Adding values

At the end : [1, 2, 3].push(4) #=> [1, 2, 3, 4]

At the start : [1, 2, 3].unshift(0) #=> [0, 1, 2, 3]

Removing values

From the end : [1, 2, 3].pop #=> [1, 2]

From the start : [1, 2, 3].shift #=> [2, 3]

Note that both Array#pop and Array#shift change the array on which they are called while returning the element that was removed.

Conclusion

That's it for the most basic uses of an array storing values. Stay tuned for advanced usage of Arrays (mostly using the mixin Enumerable).
